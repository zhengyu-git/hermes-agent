# PerfSight 产品功能介绍

## 官方描述
PerfSight是专为游戏打造性能监测闭环，能实现专业性能监控的游戏性能管理平台。

## 支持的功能

### 系统基础指标
PerfSight支持系统基础指标的监控

### 引擎特性
支持Unity、Unreal等多种游戏引擎

### 应用场景
- **游戏研发期**：辅助决策
- **游戏运营期**：性能监控

## 最新动态
- 2025年12月15日: PerfSight开放试用
- 支持系统基础指标以及引擎特性
