# SDK合规使用指南

## 概述
为帮助使用PerfSight SDK的开发运营者在符合个人信息保护相关法律法规、政策及标准的规定下合规接入、使用第三方SDK，PerfSight 特制定本指南。

## 一、接入/升级至满足监管新规的最新SDK版本
强烈建议您升级使用最新版本SDK，以便保障您正常使用SDK最新功能、避免因更新不及时产生的不利影响（例如APP被通报或下架等）。

## 二、APP隐私政策中应披露第三方SDK相关情况
请您务必明确告知终端用户您的App集成了第三方SDK服务。应在《隐私政策》中添加关于本SDK收集使用个人信息的目的、方式和范围等。

### 披露文案示例

**第三方SDK名称：PerfSight Android SDK**
- 第三方SDK提供方：公司名称：深圳市腾讯计算机系统有限公司
- 使用目的及功能场景：收集并上报玩家在游戏过程中的FPS、CPU、内存、流量、网络数据及其他自定义业务层数据
- 处理的个人信息类型：设备型号、openID、操作系统版本、Wifi状态
- 实现SDK产品功能所需的权限：网络及Wifi状态访问权限
- 第三方SDK隐私政策链接：https://perfsight.qq.com/policy
- 第三方SDK官网链接：https://perfsight.qq.com/
- 处理方式：去标识化、加密传输的安全处理方式

**第三方SDK名称：PerfSight iOS SDK**
- 第三方SDK提供方：公司名称：深圳市腾讯计算机系统有限公司
- 使用目的及功能场景：收集并上报玩家在游戏过程中的FPS、CPU、内存、流量、网络数据及其他自定义业务层数据
- 处理的个人信息类型：IDFV、设备型号、openID、Wifi状态、操作系统版本
- 实现SDK产品功能所需的权限：无
- 第三方SDK隐私政策链接：https://perfsight.qq.com/policy
- 第三方SDK官网链接：https://perfsight.qq.com/
- 处理方式：去标识化、加密传输的安全处理方式

## 三、获得用户同意后再初始化SDK
1. 确保在用户阅读APP隐私政策并取得用户授权之后，按APP功能需要在合适时机调用initContext接口初始化PerfSight SDK
2. 如果用户不同意《隐私政策》授权，则不能调用初始化接口
3. 请勿在用户同意隐私政策之前动态申请涉及用户个人信息的敏感设备权限
4. 请勿在用户同意隐私政策前私自采集和上报个人信息

## 四、可选信息配置开关
SDK向您提供了可选个人信息及权限的控制开关，您可以根据APP所需的SDK功能服务自行配置打开或关闭隐私信息请求开关。

### 1、配置可选个人信息
SDK不强制获取可选参数，即使没有获取可选参数，SDK提供的基本功能也能正常运行。

### 2、指导建议
- 未经用户同意不得收集任何个人信息
- 非服务所必需或无合理应用场景下，用户拒绝相关授权申请后，应用不得自动退出或关闭
- 不得未明确告知用户索取权限的目的和用途
- APP首次打开或运行中，未见使用权限对应的相关功能或服务时，不应提前向用户弹窗申请开启敏感权限
- 不得超出业务功能实际需要过度收集个人信息

## 五、用户权利保障机制
本SDK提供以下接口配置：
1. 终端用户请求查阅、复制其个人信息时，可通过访问PerfSight官网联系SDK团队
2. 终端用户撤销同意处理其个人信息的授权时，可通过官网联系SDK团队停止使用SDK功能
3. 终端用户请求删除其个人信息时，可通过官网联系SDK团队删除采集的用户个人信息
