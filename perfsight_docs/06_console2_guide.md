## 性能总览和自定义看板

在"性能总览"中，您可以查看项目的整体性能概况。在平台看板中，您可以通过版本/时间选取设置默认的观测条件，查看性能总览和基础指标数据。您也可以在最近访问中快速跳转历史记录，或通过自定义看板灵活配置个人看板。

### 1. 版本/时间选取

您可以在此处设置观测数据的时间维度，包括近24小时、近一周等选项。 您可以在此处设置观测数据的版本（最新运营、最新开发等），并可以点击"查看版本时间轴"查看版本发布时间线。您也可以点击"设置"按钮切换预设的版本和场景。 

### 2. 性能总览

在性能总览模块，您可以查看当前版本/场景下的总体性能情况，包括性能雷达图、FPS均值、FPS抖动、Jank卡顿、BigJank卡顿、FPS卡顿、内存、对局数等关键指标。 时间维度选择近24h时，各指标同时展示日环比和上周同期的变化情况。 时间维度选择近一周时，各指标同时展示周环比变化情况。 

雷达图统计规则

近24小时：取近24h指标均值
近一周：对7天内的指标求均值

取前30天的数据为基准，求均值和标准差，根据所处区间映射得分。

| 所处区间 | 映射得分 |
|---|---|
| 均值+1sigma——均值+3sigma | 90-100 |
| 均值-1sigma——均值+1sigma | 70-90 |
| 均值-2sigma——均值-1sigma | 60-70 |
| <均值-2sigma | 50-60(最低50分) |

90-100为优秀，70-90分为良好，60-70分为警告，低于60分为危险。

### 3. 基础指标

在基础指标模块，您可以查看当前版本/场景下的基础性能指标趋势，包括流畅度趋势、流畅度卡顿趋势、硬件趋势等。您可以点击各模块中的指标按钮（如FPS均值、FPS抖动、FPS低帧、FPS卡顿、Jank卡顿等）切换不同的趋势图。 

您可以点击"下载"按钮将数据保存至本地；点击"详情分析"可跳转至"性能分析"页面进行深入分析。

### 4. 最近访问

在最近访问模块，您可以查看近期的访问记录，包括自定义分析、性能分析等历史操作，方便快速跳转。 

### 5. 自定义看板

您可以点击"添加看板"创建自定义看板，灵活配置个人专属的数据视图。 

#### 1）新建图表

您可以点击"新建图表"创建自定义图表，配置项说明如下：

| 配置项 | 说明 |
|---|---|
| 图表设置 | 选择图表类型（折线图、柱状图、表格）、时间范围、对比方式、图表名称、指标类型（基础性能指标和自定义指标）、时间维度 |
| 查询指标 | 设置需要观测的指标，如FPS均值、场景持续时间等 |
| 筛选条件 | 选择版本、场景、机型和设置项，版本支持选择动态版本,如最新运营版本和最新开发版本 |
| 分组维度 | 选择柱状图和表格类型时，可以进行分组维度设置，选择按版本、场景、机型等维度进行分组 |

完成配置后，点击"查询"即可查看对应的图表数据；点击"保存"可保存当前配置，下次访问该看板时无需重新配置。 

#### 2）看板设置

每次进入看板时会实时刷新数据，您可以点击"新增筛选"设置二次筛选条件，包括时间范围、版本、场景、机型等。  您可以点击图标标题处的拖拽按钮，对看板内图表的顺序进行调整

## 性能分析

在"性能分析"中，您可以选择筛选条件，查看筛选条件下的性能趋势，并可以对这些数据进行版本、场景、机型等维度的下钻分析，并可以在表格中灵活查看各维度下的累计、日级、小时级数据详情。

### 1. 筛选条件

在此处您可以选择筛选条件，选择时间维度（版本周期/时间区间）、版本、场景等对数据进行筛选查询。 

### 2. 性能趋势

在性能趋势中，您可以查看筛选条件下的流畅度、硬件、场景等各类别的大盘数据，包括筛选条件下的整体大盘均值及环比变化，以及随时间变化的流水图，并下载该指标分类下的全部数据。 

您可以切换统计维度，查看日级或小时级数据；您可以切换趋势类型，包括时间趋势展示、日同比趋势展示、周同比趋势展示；也支持切换纵轴展示区间。 

### 3. 维度分析

在维度分析中，您可以对筛选条件下的数据进行各维度的下钻分析，默认提供了版本、场景、机型等单一维度的下钻分析。 

您可以在通过【新增分组维度】增加分析维度，实现多维度的交叉组合分析。您可以在列表中选择下钻维度的内容以控制各维度具体选中的内容。 

您可以点击自定义分析任意选取下钻维度，实现各维度任意组合的下钻分析。 

您可以图表形式为切换为表格和柱状图进行查看。

### 4. 数据详情

在数据详情中，您可以灵活查看各统计维度、各分类维度的性能数据。您可以选择统计维度，查看日级数据、小时数据、累计数据。

| 统计维度 | 描述 |
|---|---|
| 日级数据 | 展示筛选条件下的日级流水数据 |
| 小时数据 | 展示筛选条件下的小时级流水数据 |
| 累计数据 | 展示筛选条件下的大盘均值 |

您可以选择分组维度，对数据进行多维度交叉分组查看指标: 

Tips：旧版本的多个页面如场景分析、机型分析、SoC分析已聚合为当前的性能分析页。你可以通过分组维度的选取灵活实现原有的跨页面分析能力。 ——如需要对SoC进行分析，只需要勾选"硬件分组维度"中的"芯片"选项。

您可以对各行数据进行下钻分析：在某一行选择下钻维度，本行已有条件作为过滤条件，维度作为分组维度。表格仅展示下钻后的结果表格，上方显示下钻操作时应用的过滤条件。 

您可以点击"查看对局"跳转条件查询页，自动应用页面及本行数据作为筛选条件，查看具体对局指标。您可以点击场景名进入场景详情页，点击机型名进入机型详情页。

## 场景详情页

从性能分析页点击场景名可跳转至本页面，场景详情页包含特定场景相关数据分析，包括场景性能趋势、版本数据对比、场景下钻分布、该场景下机型性能分析及该场景下的各机型性能列表。

### 1. 场景页信息

您可以查看该场景详情页的相关信息，包含该场景详情页数据的筛选条件（从性能分析页携带），场景名称及该筛选条件下的对局数及占比，您可以在此处调整筛选条件查看对应数据。 

### 2. 场景性能趋势

您可以在此处查看该场景及筛选条件下的各性能指标的趋势，其他操作同性能分析——性能趋势 

### 3. 场景下钻分布

您可以在此处查看该场景及筛选条件下各机型、画质、画质子维度的对局分布，您可以自由选择查看的分布维度，并可以切换饼状图/列表。 

### 4. 场景版本对比

您可以在此处查看该场景已选筛选条件各版本的性能数据，并可以调整查看的指标及下载数据。 

### 5. 机型性能分布

您可以在此处查看该场景及筛选条件下的各机型性能的分布，包括各机型的FPS均值+JANK卡顿散点图。并根据这两个指标判断异常对局，展示异常对局数最高的异常机型。 

### 6. 机型性能列表

您可以在此处查看该场景及筛选条件下的各机型性能指标，列表操作同性能分析——数据详情。

## 机型详情页

从性能分析页点击机型名可跳转至本页面，机型详情页包含特定机型相关数据分析，包括机型性能趋势、机型下钻分布、相关机型分析、相关场景分析、版本数据对比、对局性能分布、对局性能列表

### 1. 机型页信息

您可以查看该机型详情页的相关信息，包含该机型详情页数据的筛选条件（从性能分析页携带），您可以在此处调整机型无关的筛选条件，如时间、版本、场景、自定义设置项，以查看不同条件下的机型性能数据。 筛选条件下方展示机型名称及信息，该筛选条件下的对局数，占比及用户数。 

### 2. 机型性能趋势

您可以在此处查看该机型及筛选条件下的各性能指标的趋势，其他操作同性能分析——性能趋势 

### 3. 机型下钻分布

您可以在此处查看该机型及筛选条件下各设置项子维度的对局分布，您可以自由选择查看的分布维度，并可以切换饼状图/列表。 

### 4. 相关机型分析

#### 1）相同SoC机型

您可以在此处查看该机型相同SoC机型的FPS均值——Jank卡顿散点分布图，并查看对应数据及排名。橙色标记为该机型详情页对应机型。 

#### 2）相同档位机型

您可以在此处查看该机型相同档位机型的相关分析，操作同上，需要业务上报机型档位信息。

### 5. 相关场景分析

您可以在此处查看该机型已选场景的性能数据，点击场景名携带条件进入场景详情页，点击机型名携带条件进入机型详情页，点击"查看对局"携带条件跳转条件查询页。 

### 6. 机型版本对比

您可以在此处查看该机型已选筛选条件各版本的性能数据，并可以调整查看的指标及下载数据。 

### 7. 对局性能分布

您可以在此处查看该机型及筛选条件下的各对局性能的分布（FPS均值——JANK卡顿散点图），右侧展示具体对局性能数据。 

您可以点击散点图中的点查看对应信息，点击"用户ID"跳转用户ID查询页。

### 8. 对局性能列表

您可以在此处查看该机型及筛选条件下的对局性能指标，列表同条件查询——数据详情。
