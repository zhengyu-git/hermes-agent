# OpenApi

## 概述
PerfSight 提供完整的 OpenApi 接口文档，供开发者调用。

## 完整接口文档
查看完整的 OpenApi 接口文档，请访问：
https://perfsight.qq.com/openapi-docs/zh/

文档包含详细的接口说明、参数定义、返回值示例等内容。

## 快速跳转
OpenAPI文档站点包含以下主要内容：
- 整体调用说明
- 通用聚合数据API
- 条件查询API
- 自定义画质API
- 网络延迟数据API
- 单局KV数据API
- 单局详情API
- 自定义模板分析API
- 机型分析API
- 场景聚合数据API
- 用户详情数据API
