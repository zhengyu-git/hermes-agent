# PerfSight 帮助文档 - 概述

## 概述

PerfSight，用大数据让性能"说话"

PerfSight是一个适用于游戏外网性能监测分析的平台，其实时采集真实玩家游戏过程中的性能数据，包括FPS、内存、CPU使用率、网络延迟、网络流量、功耗、温度、降频等指标以及卡顿堆栈，并支持高性能自定义数据的上报。PerfSight支持按时间、版本、场景、机型以及自定义画质进行精细化的统计分析展示和对比，实现游戏性能全方位监控分析，适用于从研发期到运营期的全生命周期，支持全平台（Android、iOS、Windows、Switch、XBox、PlayStation），全引擎(Unity、Unreal、Cocos2d-x、自研)。
