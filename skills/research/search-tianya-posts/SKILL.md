# 搜索天涯神帖内容

## 背景
天涯论坛已关闭，但大量神帖内容仍散落在各个转载平台。WSL环境下网络受限，需要选择合适的搜索引擎和内容源。

## 搜索引擎选择
在WSL环境下：
- ❌ Google/Bing：被墙，无法访问
- ✅ 360搜索（so.com）：可用，推荐使用
- ✅ 百度：可用，但结果可能有广告

## 内容源优先级

### 1. 今日头条（推荐）
- **优点**：通常可以直接访问，无需登录，内容较完整
- **缺点**：可能是解读版本而非原文
- **访问方式**：搜索结果中找 `www.toutiao.com` 链接

### 2. 知乎
- **优点**：内容质量高，常有原文PDF链接
- **缺点**：需要登录才能查看完整内容
- **访问方式**：搜索结果中找 `zhuanlan.zhihu.com` 链接
- **注意**：WSL浏览器环境可能无法登录

### 3. 道客巴巴
- **优点**：可能有完整文档
- **缺点**：链接可能不稳定，可能需要下载
- **访问方式**：搜索结果中找 `www.doc88.com` 链接

### 4. 哔哩哔哩
- **优点**：有视频版本，适合听书
- **缺点**：不是文字内容
- **访问方式**：搜索结果中找 `www.bilibili.com` 链接

### 5. 网盘链接
- 常见：夸克网盘（pan.quark.cn）、百度网盘
- 可能在知乎文章中找到

## 搜索步骤

1. **构造搜索词**
   - 使用完整标题搜索：`因为贱-所以生活艰辛!说出真相-底层会更绝望-大鹏金翅明王`
   - 或简化搜索：`因为贱所以生活艰辛 大鹏金翅明王`

2. **使用360搜索**
   ```
   https://www.so.com/s?q=搜索词
   ```

3. **按优先级尝试点击链接**
   - 先试今日头条
   - 再试知乎（可能需要登录）
   - 最后试道客巴巴

4. **提取内容**
   - 如果是文章页面，直接阅读总结
   - 如果是PDF链接，考虑是否需要下载

## 本地PDF检查（优先！）
在搜索之前，**先检查用户本地是否已有该帖子的PDF**：
- 用户Windows桌面有天涯神帖合集：`/mnt/c/Users/v-zhengyu002/Desktop/PDF/ty社区导入神贴/天涯神贴/`
- 子目录结构可能包含：`天涯全集（更全，但部分需解压）/天涯合集235篇/所有合集/`
- 使用 `search_files` 搜索关键词（如作者名、帖子标题片段）
- **当网络搜索渠道都被拦截时，本地PDF是最可靠的方案**

### 从本地PDF提取文本
使用 `pdf-text-extraction-wsl` 技能（pdfjs-dist Node.js库）：
1. 用 `search_files` 在本地目录搜索目标PDF
2. 用 `find` 命令获取精确路径（避免中文/空格路径问题）
3. 用 pdfjs-dist `.cjs` 脚本提取文本到 `~/.hermes/pdf_contents/`
4. 用 `read_file` 分页读取提取后的文本

**重要**：大型天涯神帖PDF可能超过2000页（如"小小茶馆"2767页/150万字），提取需要30-60秒。pdfjs-dist v4+必须用CommonJS `.cjs` + async包装函数，不能直接用ESM。

## 注意事项
- 天涯神帖可能有多个版本（原文、解读版、删减版）
- 今日头条的文章通常是解读版，不是原文
- 如果用户需要原文，本地PDF优先于网络搜索
- 总结时要说明是原文还是解读版
- 大型PDF解读时不需要逐页细读，看开头/中间/结尾抓住核心观点即可
