# 整体调用说明

## 1 签名

### 1.1 签名参数

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| access_token | string | 是 | PerfSight平台生成的token，在配置管理-产品信息中通过APPKEY获取 |
| env | string | 是 | 国内环境: v2; 新加坡环境: sgp; |
| app_id | string | 是 | 项目ID |
| username | string | 是 | 登录PerfSight平台的用户名 |

### 1.2 签名方法

以python为例:

1. 将业务接口实际参数**填入payload中**
```python
payload = {
    "platform": 0,
}
```

2. 设置token过期时间(以120秒为例)
```python
exp = int(time.time() + 120)
```

3. 把exp添加到参数中
```python
payload = {
    "platform": 0,
    "exp": exp
}
```

4. 使用jwt生成token
```python
headers = {
  "alg": "HS256",
  "typ": "JWT"
}
jwt_token = jwt.encode(payload=payload, key=access_token, algorithm='HS256', headers=headers)
```

4. 构造上传参数（请勿将业务参数直接填在这里）
```python
params = {
    'app_id': "123",
    'env': 'sgp',
    'username': 'abc',
    'token': jwt_token, //包含了业务接口实际参数
}
```

5. 发送请求
```python
headers = {'Content-Type': 'application/json'}
url = 'https://api.perfsight.qq.com/openapi/scene/dayAggregateData'
response = requests.post(url, json=params, headers=headers)
print(response.json())
```

6. 返回数据格式
```json
{
  {
  "ret": 200, // 状态码
  "msg": "", // 错误消息
  "data": [ // 返回数据
    {
      "merge_count": 581179108,
      "fps_mean": 52.56,
      "fps_swing_hour": 11.03,
      "jank_frame_hour": 5.99,
      "pss": 1425.3,
      "vss": 12507.61,
      "psmall_jank": 3.51,
      "frame100": 0.02,
      "mono": 183.34
    }
  ],
  "requestid": "94e2ac8b38b08ec238d492a708e3d023", // 服务端请求ID
  "flush_time": "2023-02-24 17:21:54" // 返回时间
}
}
```

### 1.3 请求地址

国内环境: `https://api.perfsight.qq.com`
新加坡环境: `https://api.perfsight.wetest.net`

> 不同环境情注意 env参数的给值不同 【国内环境: v2; 新加坡环境: sgp;】
