# 获取自定义画质 POST

**接口地址**: `/openapi/common/qualityCustomize`

**接口说明**: 获取当前自定义画质的配置项，一般用于其他接口的画质部分填充

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_qualityCustomize.py

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| platform | integer | 否 | 平台0: Android, 1: iOS, 5: Windows |

### 请求示例

```json
{
    "platform": 0
}
```

---

## 返回响应

**🟢200 成功**

```json
{
    "自定义画质分类列表": {
        "附加属性": "integer | number | string | boolean"
    }
}
```

> 返回数据为自定义画质分类列表，包含附加属性信息
