# 单局详情 POST

**接口地址**: `/openapi/single/detail`

**接口说明**: 获取单条对局详情，包含单条对局的整体性能数据 及 FPS、CPU 曲线等秒级数据

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_Detail.py

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| username | string | 是 | 用户名 |
| Idstr | string | 是 | 对局ID，可以通过条件查询获取 |

### 请求示例

```json
{
    "username": "string",
    "Idstr": "string"
}
```

---

## 返回响应

**🟢200 成功**

```json
{
    "property1": "string",
    "property2": "string"
}
```

> 返回数据为单局详细分析数据，包含整体性能数据及FPS、CPU曲线等秒级数据
