# 机型分析（历史接口 仅供参考） POST

**接口地址**: `/openapi/device/deviceAnalysis`

**接口说明**: 历史接口 仅供参考 新接入请使用通用聚合查询

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| platform | integer | 是 | 平台0: Android, 1: iOS, 5: Windows |
| project_version | string | 是 | 项目版本 |
| scene | string | 是 | 场景标签 |
| is_customize | boolean | 是 | 是否自定义 |
| customize_arr | array[object] | 是 | 自定义画质 |
| category | string | 是 | 分类 |
| category_items | array[string] | 是 | 分类项 |
| device_search | string | 否 | 搜索的机型列表 |
| region | string | 是 | 地区,不筛选填"world" |
| start | string | 是 | 开始时间 |
| end | string | 是 | 结束时间 |
| time_level | integer | 是 | 时间粒度 |
| device_class | array[integer] | 否 | 机型分档 |

### 请求示例

```json
{
    "platform": 5,
    "project_version": "1.101.37100.10.4.0",
    "scene": "Remote_Dam_Iris_Level1, Remote_Dam_Iris_Level2, Remote_Forrest_Level1, Remote_Forrest_Level2",
    "is_customize": true,
    "customize_arr": [
        {
            "category": "画质",
            "category_items": ["极高"]
        },
        {
            "category": "帧率",
            "category_items": ["无限制"]
        },
        {
            "category": "机型分档",
            "category_items": ["EPIC"]
        }
    ],
    "device_search":"全部",
    "region": "world",
    "start": "2025-03-10 00:00:00",
    "end": "2025-03-10 00:00:00",
    "time_level": 2,

}
```

---

## 返回响应

**🟢200 成功**

```json
{
    "property1": "string",
    "property2": "string"
}
```

> 返回数据包含机型分析的详细性能数据
