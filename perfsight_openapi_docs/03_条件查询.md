# 条件查询 POST

**接口地址**: `/openapi/single/conditionalSearch`

**接口说明**: 该接口对应页面《条件查询》功能，需要额外定制筛选项请联系小助手

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_conditionalSearch.py

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| stime | string | 是 | 开始时间 |
| etime | string | 是 | 结束时间 |
| version | array[object] | 是 | 版本 |
| version | string | 否 | 版本名称 |
| platform | string | 是 | 平台0: Android, 1: iOS, 5: Windows |
| userName | string | 是 | 用户名（调用者id） |
| size | integer | 是 | 查询数量 不超过10000 |
| source | integer | 是 | 环境 国内 1 , 国际 2 |
| timeSortField | integer | 是 | 时间排序字段 1：场景开始时间 2：场景上传时间 |
| requestid | string | 是 | 请求ID |
| model | array[string] | 是 | 机型筛选 |
| userIdList | array[string] | 是 | 查询玩家列表 |
| scene | array[string] | 否 | 场景列表 |

### 请求示例

```json
{
    "stime": "2026-01-12 00:00:00",
    "etime": "2026-01-15 23:59:59",
    "version": [
        {
            "version": "1.0.163.163.163"
        }
    ],
    "platform": "0",
    "userName": "zephyrche",
    "size": 1000,
    "source": 1,
    "timeSortField": 2,
    "model": [
        "iPhone 15 Pro",
        "SM-S9280"
    ]
}
```
