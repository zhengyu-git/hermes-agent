# 场景聚合数据（历史接口 仅供参考） POST

**接口地址**: `/openapi/scene/dayAggregateData`

**接口说明**: 历史接口 仅供参考 新接入请使用通用聚合查询

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| platform | integer | 是 | 平台0: Android, 1: iOS, 5: Windows |
| project_version | string | 是 | 项目版本 |
| scene | string | 是 | 场景名称 |
| country | string | 否 | 国家,国内环境填"world" |
| start | string | 是 | 开始时间 |
| end | string | 是 | 结束时间 |
| time_level | integer | 是 | 时间粒度,1:版本汇总,2：日级数据, 4:区间累计 |
| is_customize | boolean | 否 | 自定义画质开关，默认false |

### 请求示例

```json
{
    "platform": 0,
    "project_version": "string",
    "scene": "string",
    "country": "string",
    "start": "string",
    "end": "string",
    "time_level": 0,
    "is_customize": true
}
```
