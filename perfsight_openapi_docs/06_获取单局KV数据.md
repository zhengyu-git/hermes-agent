# 获取单局 KV 数据 POST

**接口地址**: `/openapi/single/getKv`

**接口说明**: 获取单条详情对应上报的 KV 数据，该接口需要依赖单局详情接口，请在调用该接口前调用单局详情接口

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_getKV.py

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| Idstr | string | 否 | 对局ID，可以通过条件查询获取 |
| UserName | string | 否 | 用户名 |
| enableEventKv | string | 是 | 是否需要 KV，填写true |

### 请求示例

```json
{
    "Idstr": "string",
    "UserName": "string",
    "enableEventKv": "string"
}
```

---

## 返回响应

**🟢200 成功**

```json
{
    "EventKv": [
        {
            "property1": "string",
            "property2": "string"
        }
    ],
    "fpsTime": [
        "string"
    ]
}
```

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| EventKv | array[object] | 自定义KV事件列表 |
| fpsTime | array[string] | FPS时间轴 |
| 附加属性 | integer \| number \| string \| boolean | 其他附加属性 |
