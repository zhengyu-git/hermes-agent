# 获取自定义模板分析 POST

**接口地址**: `/openapi/common/viewReportData`

**接口说明**: 获取自定义模板分析数据

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_viewReportData.py

---

## 请求参数

**Query 参数**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| app_id | string | 否 | 应用ID |
| username | string | 否 | 用户名 |
| token | string | 否 | 生成的JWT字符串 |

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| exp | integer | 是 | 过期时间戳 |
| viewName | string | 是 | 视图名称 |
| version | array[string] | 是 | 版本列表 |
| aggres | array[string] | 是 | 聚合维度 |
| archive | string | 是 | 归档类型 |
| filters | object | 是 | 过滤器 |
| app_version | array[string] | 是 | 应用版本 |
| scene_name | array[string] | 是 | 场景名称 |
| isCustomize | boolean | 是 | 是否自定义 |
| customizeClassArr | array[object] | 是 | 自定义分类数组 |
| category | string | 是 | 分类 |
| category_items | array[string] | 是 | 分类项 |
| isPeriod | boolean | 是 | 是否周期 |
| stime | string | 是 | 开始时间 |
| etime | string | 是 | 结束时间 |
| platform | integer | 是 | 平台 |
| source | integer | 是 | 来源 |
| requestid | string | 是 | 请求ID |
| userName | string | 是 | 用户名 |

### 请求示例

```json
{
  "exp": 1776064394,
  "viewName": "自定义指标均值统计",
  "version": ["1.101.37110.20.7.0"],
  "aggres": [],
  "archive": "year",
  "filters": {
    "app_version": ["1.101.37110.20.7.0"],
    "scene_name": ["Lobby", "Battle"]
  },
  "isCustomize": true,
  "customizeClassArr": [
    {
      "category": "帧率",
      "category_items": ["240"]
    },
    {
      "category": "画质",

    }

}
```
