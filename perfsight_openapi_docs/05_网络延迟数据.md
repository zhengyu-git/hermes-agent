# 网络延迟数据 POST

**接口地址**: `/openapi/net/netDelayData`

**接口说明**: 获取国家、省市等维度的网络延迟数据

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_netDelayData.py

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| RequestID | string | 否 | 请求ID |
| start | string | 是 | 开始时间 |
| AggType | string | 是 | 聚合类型: "country" - 按国家聚合; "province" - 按省份聚合; "city" - 按城市聚合; "isp" - 按运营商聚合; "network" - 按网络类型聚合 |
| end | string | 是 | 结束时间 |
| Platform | integer | 是 | 平台0: Android, 1: iOS, 5: Windows |
| _country | array[string] | 否 | 国家名称过滤 |
| _province | array[string] | 否 | 省份名称过滤，不过滤填"全部" |
| _city | array[string] | 否 | 城市名称过滤，不过滤填"综合" |
| app_version | array[string] | 否 | APP版本过滤 |
| _scene | array[string] | 否 | 场景名称过滤 |
