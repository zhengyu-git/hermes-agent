# 用户详情数据获取（历史接口 仅供参考） POST

**接口地址**: `/openapi/single/detailQuery`

**接口说明**: 历史接口 仅供参考 新接入请使用条件查询

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| platform | enum<integer> | 否 | 平台ID (0: Android, 1: iOS, 5: Windows) |
| requestid | string | 是 | 请求ID，UUID格式 |
| user_id | string | 是 | 用户ID |
| apmEnv | enum<string> | 是 | 站点区域 (v2: 国内, sgp: 新加坡) |
| stime | string | 是 | 开始时间 (格式: yyyy-MM-dd HH:mm:ss) |
| etime | string | 是 | 结束时间 (格式: yyyy-MM-dd HH:mm:ss) |
| timeSortField | enum<integer> | 是 | 时间排序字段 (1: 场景开始时间, 2: 场景上传时间) |
| size | integer | 是 | 查询条数 (1-10000, 默认1000) |

### 请求示例

```json
{
    "user_id": "2315dc13e13c1eb5",
    "timeSortField": 1,
    "apmEnv": "v2",
    "platform": "0",
    "requestid": "3b7baf973ed5cbe17d9f5d4cb7eaceb4",
    "stime": "2024-02-28 00:00:00",
    "etime": "2024-02-28 23:59:59",
    "size": 1
}
```
