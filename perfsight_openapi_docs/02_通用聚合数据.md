# 通用聚合数据 POST

**接口地址**: `/openapi/common/aggregateData`

**接口说明**: 获取聚合数据的通用接口，支持筛选条件、聚合维度等自定义设置

**示例脚本**: https://perfsight-docs-1258344700.cos.ap-nanjing.myqcloud.com/open/PS_aggregateData.py

---

## 请求参数

**Body 参数 (application/json)**

| 参数名称 | 类型 | 必选 | 描述 |
|---------|------|------|------|
| version | array[string] | 是 | 版本名称 |
| sceneName | array[string] | 是 | 场景名称 |
| isCustomize | boolean | 是 | 是否支持自定义画质，true 的话需要传入 customArr |
| customizeArr | array[object] | 是 | 自定义画质项 |
| CategoryId | integer | 否 | CategoryId |
| CategoryItemId | integer | 否 | CategoryItemId |
| country | array[string] | 是 | 国家 |
| aggres | array[string] | 是 | 聚合维度，支持按照{app_version、scene_name、model、soc_gpu、cpu_name、gpu_name、device_class}聚合，不填默认按时间聚合 |
| archive | string | 是 | all:版本周期 week:周级 day:日级 hour:小时级 interval:区间累计 |
| stime | string | 是 | 开始时间 |
| etime | string | 是 | 结束时间 |
| platform | integer | 是 | 平台0: Android, 1: iOS, 5: Windows |
| userName | string | 是 | 调用人用户ID |

### 请求示例

```json
{
  "version": [
    "0.0.0.0.8627821(Publish_Marketing_Android_Dev.#185)"
  ],
  "sceneName": [ "迷雾荒岛"],

  "isCustomize": true,
  "customizeArr": [
    {
      "CategoryId": 5,
      "CategoryItemId": 2
    }
  ],
  "country":["world"],
  "aggres": ["app_version"],
  "archive": "year",  //all:版本周期 week:周级 day:日级 hour:小时级 interval:区间累计
  "stime": "2025-12-19 00:00:00",
  "etime": "2025-12-19 23:59:59",
  "platform": 0,
  "requestid": "20370fc56eda438daf7d139101ad188e",
  "userName": "willsondang"
}
```
